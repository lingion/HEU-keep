# keep

#### 介绍
keep运动截图生成

#### 说明

1.  本项目魔改自项目[南邮keep跑步打卡截图修改](https://gitee.com/jimaske/njupt-keep).
2.  原作者: 四脚兽起飞([@jimaske](https://gitee.com/jimaske)); 由"风云de寒冰"([@fyhb](https://gitee.com/fyhb))魔改并发布.

#### 相比于原项目的改动

1.  增加了修改截图中头像与背景图片的功能.
2.  增加了加载网页时自动随机生成公里数与配速的功能.
3.  增加了加载网页时自动填充当前时间的功能.
4.  高清重制了截图中的界面部分.
5.  微调了截图中部分文字的组织形式及排布.
6.  改动了网页的部分界面.
7.  使用原生javascript重构了整体代码(真的只是因为我比较无聊, 并不是因为我不会Vue什么的👋).
8.  迎合贡🍊🍊跑的需要对背景素材进行了替换.

#### 使用说明

1.  修改"数据设置"部分的输入框中的内容即可更改截图中出现的各项数值.
2.  点击"保存图片"即可自动生成截图并下载到本地.
